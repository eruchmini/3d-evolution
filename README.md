# Simple 3D Game

A basic 3D game built with Three.js where you control a cube and collect golden spheres!

## How to Play

1. Open `index.html` in a web browser
2. Use the controls to move around and collect golden spheres
3. Avoid obstacles and try to collect all the spheres!

## Controls

- **W** or **↑** - Move forward
- **S** or **↓** - Move backward
- **A** or **←** - Move left
- **D** or **→** - Move right
- **Space** - Jump

## Features

- 3D graphics using Three.js
- Smooth camera following
- Physics-based movement with momentum
- Gravity and jumping
- 10 collectible golden spheres
- 15 random obstacles
- Score tracking
- Real-time speed display

## Technical Details

- Built with Three.js v0.160.0
- Uses ES6 modules
- Single HTML file (no build process required)
- Responsive canvas that fits the browser window

## Deployment

This game is configured to deploy on Cloudflare Pages using the `wrangler.jsonc` configuration.

Simply push to your repository and Cloudflare Pages will automatically deploy the game!

Enjoy the game!
